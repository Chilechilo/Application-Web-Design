## Markdown

**¿Que es Markdown?**

Es un lenguaje que facilita la aplicacion de formato a un texto, empleando caracteres de manera especial. Se penso para crear textos los cuales estarian destinados para la web con mas rapidez y sencillez a que estuvieramos implementado HTML directamente.

Markdown como lo explica John Gruber, es dos cosas:
- El lenguaje
- Herramienta que convierte el lenguaje en HTML valido

**Diferentes usos**

- Escribir web es mas rapido y comodo
- Es mas dificil cometer errores de sintaxis
- Es ideal para entornos moviles
